The project focuses on mastering **core web technologies without using front-end frameworks**.

---

# 📌 Overview

**MeinKraft Store** is a small e-commerce style web application where users can browse Minecraft items and manage a shopping cart.

The project was inspired by my son, who is passionate about **Minecraft**.
It became an opportunity to build something meaningful while strengthening my understanding of **how the web works under the hood**.

Unlike many modern projects, this store intentionally avoids frameworks like React or Vue in order to focus on **fundamental browser APIs and DOM manipulation**.

---

# 🧠 Project Philosophy

> “Beyond the store's functionality, my primary goal was to intentionally step away from frameworks like React to master the fundamental building blocks of the web—working directly with the language as it was designed.”

This project focuses on **learning the foundations of web development**:

* Understanding how the **DOM works**
* Managing state without frameworks
* Structuring scalable **TypeScript code**
* Working with **Local Storage**
* Building UI interactions using pure JavaScript

Although modern production applications often rely on frameworks, mastering the fundamentals provides a **stronger technical foundation for any developer**.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/your-username/mein-kraft-store.git
cd mein-kraft-store
```

Install dependencies

```bash
npm install
```

---

# ▶️ Running the Project

Start the development server

```bash
npm start
```

Compile TypeScript

```bash
npm run build
```

Watch SASS files

```bash
npm run sass
```

The project will run locally using **Lite Server**.
